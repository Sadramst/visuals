## [1.0.0] - 2026-05-06

### Added
- Initial release of Mine Production Gantt visual
- Power BI custom visual with full capabilities
- React component for web embedding
- Support for AM/PM/Night shift breakdown
- Target comparison with colour-coded performance
- Interactive bar selection with detail card
- Rich tooltips with crew, equipment, and location data
- Responsive design for different viewport sizes
- Animation on initial render
- Configurable colours via formatting pane
- Data labels with variance percentages
